# TechCorp-Dashboard-for-a-Fictional-Freelance-Client
TechCorp-Dashboard-for-a-Fictional-Freelance-Client
📊 Leveling up my React skills—TechCorp!

As part of my internship journey with Elevvo Pathways, I’ve successfully built a multi-page dashboard for a fictional freelance client using React.js, React Router, TailwindCSS, and Recharts.

🔹 Key Features:

📌 Overview Page: Displays total projects, earnings, tasks due, recent activity list & quick stats.

📌 Projects Page: Shows a structured list of client projects with project name, status, and deadline in a table/card layout.

📌 Profile Settings Page (Optional): User can view/edit profile info (name, email, and password placeholder fields).

🔹 Additional Implementations:

🖥 Sidebar + Top Header for consistent navigation across all pages.

🔄 Smooth routing using React Router (no page refreshes).

📈 Charts & Visuals with Recharts to display insights like earnings and task type distributions.

📱Fully responsive UI for both desktop and mobile.

🔔 Bonus: Added a notification dropdown in the top header showing the 3 most recent user activities (mock data).

🛠 Tech Stack Used:

React.js | React Router | Tailwind CSS | Recharts

➡ This project helped me strengthen concepts of multi-page layouts, reusable components, state management, routing, and responsive design. Proud to mark it as Task 6 in my internship journey at Elevvo Pathways 🚀
